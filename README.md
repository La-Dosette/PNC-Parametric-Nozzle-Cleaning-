# PNC-Parametric-Nozzle-Cleaning-
PNC (Parametric Nozzle Cleaning) is a universal nozzle cleaning system for Klipper-based 3D printers. It provides multiple parametric cleaning patterns (SAW, W, BOX) with independent control of geometry, speed and contact time. Machine-agnostic, safe by design and easy to calibrate, PNC focuses on predictable motion and clean configuration cleancfg
